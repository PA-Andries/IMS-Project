Key insight

Raw data is mostly noise

Whole-series spread (std)≈ 2.2 °C
Typical step between two readings≈ 2.6 °C
The jump between two readings ≈ the whole series' spread → no clean outliers to find on raw data.
Note  Step-to-step noise is as big as the entire signal — the raw stream is noise-dominated.

Sensitivity

How sensitive is it?

  • Test: add 12 fake spikes, count how many the detector catches.
  • Threshold k = how strict the detector is.
  • Lower k → catches more, but more false alarms.
    Higher k → safer, but misses some.
False alarms stayed below 0.5% at every setting.
Fig. 3 — Catch rate vs strictness (k).

Takeaways

In short

  • Raw 10-min stream is noise-dominated — "nothing found" is the right answer.
  • Aggregate first; use rolling context for streams.
  • The threshold sets the recall ↔ false-positive trade-off.
  • Raw data stored on AWS S3 — fully reproducible.
